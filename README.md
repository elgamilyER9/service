# مشروع الخدمات (Service Application)

عشان تشغل المشروع عندك من غير صداع، امشي ورا الخطوات دي واحدة واحدة والدنيا هتمشي معاك زي السكينة في الحلاوة.

---

## 📋 أول حاجة: حاجات لازم تكون عندك

قبل ما تبدأ، اتأكد إن جهازك عليه الحاجات دي يا كبير:

1.  **PHP** (نسخة 8.1 أو أحدث)
2.  **Composer** (عشان مكاتب الـ PHP)
3.  **Node.js & NPM** (عشان الفرونت اند)
4.  **MySQL** (قاعدة البيانات بتاعتك)

---

## 🛠 التسطيب (Installation Guide)

### 1. نزل المشروع عندك (Clone)
أول حاجة، افتح التيرمينال وهات المشروع عندك على الجهاز:
```bash
git clone <your-repository-url-here>
cd services
```

### 2. ظبط الباك اند (Laravel Setup)

1.  **نزل مكاتب الـ PHP**:
    اكتب الكود ده عشان يحملك كل اللي المشروع محتاجه:
    ```bash
    composer install
    ```

2.  **ملف الإعدادات (.env)**:
    المشروع بيبقى جاي معاه ملف اسمه `.env.example`، إحنا محتاجين نعمل واحد زيه بس اسمه `.env` عشان ده اللي هنكتب فيه إعداداتنا.
    ```bash
    cp .env.example .env
    ```
    *(أو ممكن تدخل الفولدر بنفسك وتعمل copy paste للملف وتغير اسمه لـ `.env`)*

3.  **مفتاح المشروع (App Key)**:
    لازم نولد مفتاح تشفير للمشروع، اكتب الأمر ده:
    ```bash
   ؤ
    ```

4.  **ظبط الداتا بيز**:
    افتح ملف `.env` اللي لسه عاملينه، ودور على الجزء بتاع الداتا بيز وظبطه على حسب جهازك:
    ```ini
    DB_CONNECTION=mysql
    DB_HOST=127.0.0.1
    DB_PORT=3306
    DB_DATABASE=service_app_db  # روح اعمل داتا بيز بالاسم ده في الـ MySQL الأول
    DB_USERNAME=root
    DB_PASSWORD=              # لو عامل باسوورد للداتا بيز اكتبه هنا
    ```

5.  **الجداول والبيانات (Migrations & Seeds)**:
    عشان نبني جداول الداتا بيز ونحط فيها شوية بيانات تجريبية (زي الأدمن واليوزرز)، شغل الأمر ده:
    ```bash
    php artisan migrate:fresh --seed
    ```

6.  **ظبط ملفات الصور (Storage Link)**:
    عشان الصور المرفوعة تظهر صح، لازم نعمل اللينك ده:
    ```bash
    php artisan storage:link
    ```

### 3. ظبط الفرونت اند (React Setup)

1.  **نزل مكاتب الجافاسكريبت**:
    بسيطة خالص، اكتب الأمر ده:
    ```bash
    npm install
    ```

## 🏃‍♂️ يلا نشغل المشروع

عشان المشروع يشتغل كله تمام، محتاجين نفتح **شاشتين تيرمينال (Terminals)** ونشغل فيهم أمرين مختلفين في نفس الوقت.

#### التيرمينال الأول: شغل الباك اند
```bash
php artisan serve
```
*ده هيشغل سيرفر لارافيل، وغالباً هيكون على الرابط `http://localhost:8000`*

#### التيرمينال التاني: شغل الفرونت اند
```bash
npm run dev
```
*ده هيشغل الـ Vite عشان يجمع ملفات الـ React ويحدثها أول بأول.*

✨ **كده مبروك عليك يا معلم!** افتح المتصفح بتاعك وادخل على الرابط اللي ظهرلك (غالباً `http://localhost:8000`) وهتلاقي الابليكيشن شغال معاك زي الفل.

## 📚 للناس بتوع الـ API (Developers)

لو عايز تجرب الـ API لوحدها من غير الفرونت:
*   هتلاقي ملف اسمه `service.postman_collection.json` موجود برا مع الملفات.
*   ادخله على برنامج **Postman** (Import) وهتلاقي كل الـ Endpoints جاهزة بتفاصيلها عشان تجرب براحتك.

## ❓ لو قابلتك مشاكل (Troubleshooting)

*   **مشكلة "Vite manifest not found"**:
    *   اتأكد إنك مشغل `npm run dev` في تيرمينال لوحده.

*   **مشكلة في الاتصال بالداتا بيز (Connection refused)**:
    *   اتأكد إن برنامج الـ MySQL (أو XAMPP/WAMP) شغال.
    *   راجع اليوزر والباسورد في ملف `.env`.

*   **مشاكل الصلاحيات (Permission denied)**:
    *   لو شغال على Linux/Mac ممكن تحتاج تدي صلاحيات للكتابة، جرب الأمر ده:
        ```bash
        chmod -R 775 storage bootstrap/cache
        ```

*   **مشاكل في Composer**:
    *   لو قالك الميموري خلصت (Memory limit)، جرب الأمر ده: `COMPOSER_MEMORY_LIMIT=-1 composer install`.
